Name

enable_chart -- Install chart methods for use

Synopsis

bool enable_chart ( )

Description

This method make the chart methods available for use. An implicit call to clearchart() is made when this function is called.

Examples

See Also

  • chart->clearchart()
  • chart->setcolor()
  • chart->add_series()
  • chart->place_chart()

History

This function was added in version 2.1
